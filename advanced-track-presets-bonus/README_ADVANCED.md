
Advanced Tracks — Datasets, Presets, and Snippets
=======================================================
> Author : Badr TAJINI - Machine Learning & Deep learning 2 - ECE 2025/2026

This pack adds **one intermediate** and **one difficult** dataset per track. Each preset mirrors the starter repos’ config formats, so you can swap configs without code changes.

**Starters required**
- Track A (CV classification): `cv-project-starter.zip`
- Track B (Object detection): `od-project-starter.zip`
- Track C (NLP classification): `nlp-project-starter.zip`
- Track D (Time-series/Tabular regression): `ts-project-starter.zip`

Use the per-track READMEs below and the YAML presets in `configs/`.
